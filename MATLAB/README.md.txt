Core MATLAB routines for the following analyses:

- contrastive PCA for extracting rest1 and rest2 specific components of activity
- decoding locations from contrastive subspaces
- attributing predictions to individual neurons

Required codebases:
1) generalized contrastive PCA:
https://github.com/SjulsonLab/generalized_contrastive_PCA

2) attribution analysis - integrated gradients in Gaussian process regression
https://github.com/KurtButler/2024_attributions_paper

3) buzcode
https://github.com/buzsakilab/buzcode
